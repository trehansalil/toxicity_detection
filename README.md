```bash

```

```bash
source env/bin/activate
```
- Installing Requirements file
```bash
pip install -r requirements.txt
```
- Gcloud Setup for Linux
```bash
bash gcloud_installer.sh
```
- Gcloud CLI Initialize
```bash
gcloud init
```
